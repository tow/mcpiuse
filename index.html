Model Context Protocol (MCP) is an open standard for connecting AI assistants to external data sources and tools. This site tracks which IDEs and AI assistants support MCP, and which MCP features they implement.

Not all MCP clients are equal — some support tools but not resources, others lack sampling or elicitation. Use these tables to find out what works in your environment before building or adopting MCP servers.
